# 🎓 trAcket LMS

Plateforme de Learning Management System.

## Installation

1. Cloner le projet
2. Importer `sql/install.sql` dans MySQL
3. Configurer `config/database.php`
4. `composer install`
5. Lancer `php -S localhost:8888`
